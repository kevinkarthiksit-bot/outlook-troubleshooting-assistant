STEP 1 — UNZIP (if you received a zip / .zip.txt file)
-------------------------------------------------------
Rename  .zip.txt  to  .zip  if needed, then Extract All.
Keep all subfolders together.


STEP 2 — ONE CLICK TO START (local)
------------------------------------
Double-click:

    Launch Outlook Assistant.bat

The server starts and your browser opens to Case Setup automatically.
Leave the black window open while you use the app.


STEP 3 — CASE SETUP
-------------------
1. Enter Chat IMS, Platform, and Environment.
2. Continue to the Org KB or Troubleshooting Guides.


ADMIN — UPLOAD KB (JSON or CSV)
--------------------------------
1. Open Admin dashboard (link from app or admin-login.html).
2. Sign in: Hannah / Hannah@95
3. Choose a JSON or CSV file, then click **Apply KB update**.
   (In Excel: Save As → CSV with Number + Short description columns.)


OTHER PAGES (after server is running)
--------------------------------------
  Case setup ........... http://localhost:8080/case.html
  Org KB search ........ http://localhost:8080/index.html
  Troubleshooting ...... http://localhost:8080/troubleshooting.html
  Admin dashboard ...... http://localhost:8080/admin-login.html


TO STOP THE APP
---------------
Close the black command window, or press Ctrl+C in that window.
