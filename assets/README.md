# Image assets for Outlook Troubleshooting Assistant

Step screenshots live here and are referenced from `data/kb-articles.sample.json` using paths like `assets/images/send-receive-update.svg`.

## For IT administrators

1. Upload the entire `assets/` folder alongside HTML/JS when deploying to SharePoint Document Library.
2. Replace placeholder SVG files with real PNG or JPG screenshots (same filename, or update KB JSON paths).
3. See `manifest.json` for the list of images and what each step illustrates.
4. Keep paths **relative** (e.g. `assets/images/my-screenshot.png`) so they work on SharePoint without code changes.

## Placeholder format

Current files are branded SVG placeholders generated by `node scripts/generate-placeholders.js`. They validate layout in the guide before real screenshots are available.

## Recommended captures

| File | Capture |
|------|---------|
| `send-receive-update.svg` | Outlook Send/Receive tab with Update Folder |
| `account-settings-mailbox.svg` | File > Account Settings > Advanced > Add mailbox |
| `new-outlook-toggle.svg` | Classic/New Outlook toggle in title bar |
| `credential-manager.svg` | Windows Credential Manager Office entries |
| `safe-mode-run.svg` | Run dialog with `outlook.exe /safe` |

## SharePoint permissions

Ensure all users have **Read** access to the Document Library containing `assets/images/`.
